# Rocket-Game
A 2D rocket Game created with Pygame. Featuring custom music, graphics, and keyboard controls.

Controls:
  - `WASD` / Arrow Keys to move
  - `Space` to Start / Pause the Game
  - `m` to mute sounds
    
Modules Used:
  - pygame ( Install it using `pip install pygame` )
  - time
  - os
  - random
